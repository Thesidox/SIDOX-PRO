## 🚀 Démarrage rapide

```bash
# 1. Copier les variables d'environnement
cp .env.example .env

# 2. Installer les dépendances (serveur + client)
npm run install-all

# 3. Lancer en mode développement
npm run dev
```

| URL | Description |
|-----|-------------|
| http://localhost:3000 | Interface React |
| http://localhost:5000 | API Express |
| http://localhost:3000/admin | Panel Admin |
